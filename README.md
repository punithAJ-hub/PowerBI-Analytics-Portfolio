# 📊 Power BI-Style Analytics Dashboards

An interactive business analytics portfolio built with **Chart.js** — replicating the look, feel, and interactivity of Power BI dashboards in pure HTML/CSS/JavaScript. No backend required.

🔗 **[Live Demo](https://punithaj-hub.github.io/PowerBI-Analytics-Portfolio/)**

---

## Dashboards

### 1. Sales Performance
Track monthly revenue trends, regional breakdowns, top products, and category sales mix. Filters: Year, Region, Category, Segment.

### 2. HR Analytics
Workforce insights covering headcount, attrition rates, age distribution, and key drivers of employee turnover. Filters: Department, Job Role, Gender, Age Group.

### 3. Budget vs Actuals
Financial performance tracker comparing budgeted vs actual spend across 6 cost centers, with variance analysis and quarterly drill-down. Filters: Year, Quarter, Department.

### 4. Customer Churn
Retention analytics with churn rate by contract type, monthly churn trends, and tenure-based risk segmentation. Filters: Contract Type.

### 5. Supply Chain
End-to-end supply chain view covering stock levels, supplier lead times, order fulfillment rates, and inventory health. Filters: Category, Supplier, Warehouse, Stock Status.

---

## Features

- **Interactive filters** — dropdowns update all charts in real time
- **Tab navigation** — seamless switching between 5 dashboards
- **KPI cards** — key metrics with trend indicators on each dashboard
- **Responsive design** — works on desktop and tablet
- **Power BI aesthetic** — dark sidebar, card layout, Microsoft-style color palette

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Charts | [Chart.js 4.x](https://www.chartjs.org/) |
| Layout | HTML5 / CSS3 (CSS Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES6+) |
| Hosting | GitHub Pages |
